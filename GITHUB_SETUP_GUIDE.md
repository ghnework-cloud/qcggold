# 📱 QCG Gold — GitHub Pages Setup Guide
## Step by Step in Simple English (Mobile Friendly)

---

## ✅ STEP 1 — Create GitHub Account
```
1. Open browser on phone or computer
2. Go to: github.com
3. Click "Sign up"
4. Enter your email
5. Create a password
6. Verify your email
7. Account created! ✅
```
**Cost: FREE**

---

## ✅ STEP 2 — Create New Repository
```
1. After login, click green "New" button
   (top left, looks like a book icon)

2. Fill in:
   Repository name: qcggold
   Description: QCG Gold Website
   Visibility: ✅ Public (important!)
   
3. Click "Create repository"
```

---

## ✅ STEP 3 — Upload Your Files
```
1. On the new page, click:
   "uploading an existing file"

2. Drag and drop ALL these files:
   ✅ index.html
   ✅ shop.html
   ✅ goldsave.html
   ✅ calculator.html
   ✅ about.html
   ✅ style.css
   ✅ prices.js
   ✅ main.js
   ✅ README.md

3. Scroll down, click "Commit changes"
   (green button)

4. Wait 30 seconds for upload
```

---

## ✅ STEP 4 — Enable GitHub Pages
```
1. Click "Settings" tab (top of page)

2. Scroll down left menu, click "Pages"

3. Under "Source":
   Select: Deploy from a branch
   
4. Under "Branch":
   Select: main
   Select: / (root)
   
5. Click "Save"

6. Wait 2-3 minutes
```

---

## 🔄 HOW TO UPDATE PRICES WEEKLY

```
1. Go to github.com → your repository

2. Click on file: prices.js

3. Click pencil icon ✏️ (Edit this file)

4. Find the price you want to change
   Example: find "buy: 15765" for 1g coin
   Change to new Elite Gold rate

5. Scroll down
   Click "Commit changes"
   Click "Commit changes" again

6. Wait 2 minutes
   Website automatically updates!
```

## 🌐 ADD CUSTOM DOMAIN (Optional)

If you want www.qcggold.in instead of github URL:

```
1. Buy domain at godaddy.com (₹500/year)
   Search: qcggold.in

2. In GoDaddy DNS settings, add:
   Type: CNAME
   Name: www
   Value: yourusername.github.io

3. In GitHub Pages settings:
   Custom domain: qcggold.in
   Click Save
   
4. Wait 24 hours
   Website live on qcggold.in ✅
```
